# Tools（实用工具箱）

`Tools` 是一个面向生存与联机体验优化的实用模组（Minecraft 1.21.11 + Fabric），把常用的“省操作”功能整合到一起。

## 主要功能

- **自定义堆叠**：提升物品堆叠上限，减少整理背包和仓库的时间。
- **聚合箱子（新方块）**：集中管理周边容器物品，支持搜索、排序和按量提取。
- **自动拾取**：范围内掉落物自动进背包，不用来回跑着捡。
- **经验吸收**：经验优先直接给玩家，减少经验球堆积，同时保留经验修补（Mending）效果。
- **手持潜影盒打开**：潜影盒无需放置即可直接打开管理。
- **漏斗速度调节**：按需加快或减慢漏斗物流节奏。
- **更好的村民交易**：提供更顺手的交易模式（含无限交易模式和取货上限控制）。
- **大数字显示优化**：高堆叠数量在 UI 中显示更清晰，不易遮挡。

## 安装

1. 安装对应版本的 **Fabric Loader** 与 **Fabric API**。  
2. 将 `stacksizeconf-*.jar` 放入 `.minecraft/mods`。  
3. 启动游戏即可使用。

## 配置方式

- 游戏内按键打开配置（默认快捷键：`O`，可在按键设置中修改）。
- 安装 Mod Menu 时，也可在模组列表里进入配置页面。

## 联机说明

- 该模组多数功能由服务端控制。  
- 联机时建议服务端与客户端使用同一版本，避免配置不一致带来的表现差异。
